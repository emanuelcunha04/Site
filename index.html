Introdução ao HTML

HTML (Hypertext Markup Language) é a linguagem de marcação usada para criar páginas da Web. Ele fornece uma estrutura para a criação de documentos que podem ser exibidos em um navegador da Web. A marcação HTML é usada para definir a estrutura e o conteúdo da página, incluindo texto, imagens, vídeos e links. O !DOCTYPE html é uma declaração que informa ao navegador que o documento é escrito em HTML. O html é a marcação que define o início do documento HTML. O head é o elemento que contém informações sobre o documento, como o título da página. O body é onde todo o conteúdo da página é definido. A marcação HTML é composta por tags ou elementos. As tags são usadas para marcar o início e o fim de um elemento, e o conteúdo fica entre as tags. Existem muitas tags HTML diferentes, cada uma com uma finalidade específica. Algumas das tags mais comuns incluem: h1, h2, h3, h4, h5, h6: Usadas para criar cabeçalhos de diferentes tamanhos.
p: Usado para criar parágrafos de texto.
a: Usado para criar links para outras páginas ou recursos na web. img: Usado para inserir imagens na página. ul e ol: Usado para criar listas com marcadores ou numeração. div e span: Usados para agrupar elementos juntos para aplicar estilos CSS. Além das tags, os elementos HTML também podem ter atributos que fornecem informações adicionais sobre o elemento.
